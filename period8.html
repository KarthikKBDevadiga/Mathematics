Example 1

Factorise x2 + 8x + 16

Solution: Observe the expression; it has three terms. Therefore, it does not fit Identity III. Also, it’s first and third terms are perfect squares with a positive sign before the middle term. So, it is of the form a2 + 2ab + b2. where a = x and b = 4

x2 + 8x + 16
x2 + 2 × 4 × x + 4 2
(x + 4)2

Example 1

Factorise x2 + 5x + 6

If we compare the R.H.S. of Identity (IV) with x2 + 5x + 6, we find ab = 6, and a + b = 5. From this, we must obtain a and b. The factors then will be (x + a) and (x + b).

If a b = 6, it means that a and b are factors of 6. Let us try a = 6, b = 1. For these values a + b = 5, and not 5, So this choice is not right. Let us try a = 2, b = 3. For this a + b = 5 exactly as require

x2 + 5x + 6
x2 + 2x + 3x + 6
x(x + 2) + 3(x + 2)
(x + 3)(x + 2)

Example 2

Factorise y2 + 7y + 12

If we compare the R.H.S. of Identity (IV) with y2 + 7y + 12, we find ab = 12, and a + b = 7. From this, we must obtain a and b. The factors then will be (y + a) and (y + b).

If a b = 12, it means that a and b are factors of 12. Let us try a = 3, b = 4. For these values a + b = 7

y2 + 7y + 12
y2 + 3y + 4y + 12
y(y + 3) + 4(y + 3)
(y + 4)(y + 3)
